The line...

    <script src="skidata.js"></script>

    Imports an external JavaScript (JS) file.
    In this case it's an object - similar to, but NOT actually JSON

    To check this in your code
    console.log(skidata) will show you the data


    console.log(typeof skidata) will tell you (in this case) the variable is an object.

Import JSON

    This gives an error as it's trying to import a JS object...

    <script src="skidata.json"></script>
